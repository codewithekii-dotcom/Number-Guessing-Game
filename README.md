# Number Guessing Game

My first Python project. A simple number guessing game with single player and multiplayer modes, plus difficulty levels.

## Features
- Single player or 2-player mode
- 3 difficulty levels: Easy (1-10), Medium (1-50), Hard (1-100)
- Input validation
- Hints (Too High / Too Low)

## How to Run

```bash
python number_guessing_game.py

```
## What i learned


- `input()` and `int()` for user input
- `while` loops for validation
- `if/elif/else` for menu choices
- `random.randint()` for generating random numbers
- `for` loops for limited chances
